# מדריך הסגנון של יובל — שפה גרפית Binary Code (רטרו-פיקסל)

> מדריך זה מגדיר את **השפה הוויזואלית** של כל התמונות שיובל מפיק ב-`yuval/outputs/`.
> המטרה: עקביות ויזואלית מוחלטת — כל תמונה נראית כאילו יצאה מאותו עולם עיצובי.
> כל prompt שיובל שולח ל-`gpt-image-gen` חייב לשקף את הכללים שכאן.
>
> האסתטיקה: מחשבי שולחן רטרו משנות ה-80/90, מסכים פיקסליים, מסגרות שחורות עבות,
> צורות גיאומטריות צבעוניות מפוזרות, צבעים שטוחים בניגוד גבוה.

## פלטת צבעים (חובה — בדיוק הגוונים האלו)

יש לנסח בכל prompt את הצבעים בשמותיהם ובערכי ה-HEX שלהם, כדי לקבע את הפלטה:

| תפקיד | HEX | שימוש בתמונה |
|-------|-----|---------------|
| שחור | `#000000` | רקע ראשי, מסגרות, קווי מתאר, טקסט |
| כמעט-שחור | `#1A1A1A` | מסגרות, צללים קשיחים, קווים כהים |
| ורוד חם | `#E84477` | מבטא — עיגולים, נקודות, צורות בולטות |
| טורקיז | `#3BBFB8` | מסכי מחשב, רקעים, אובייקטים קרירים |
| צהוב | `#F5C800` | מקלדות, רצועות, אלמנטים בולטים |
| צהוב רך | `#F5E070` | מקשים, פרטים משניים |
| קרם | `#F5F0D0` | רקע מסך, משטחי תוכן, רקע ניטרלי |
| אפור | `#C8C8C8` | כפתורים, פרטי מתכת/פלסטיק |

**עיקרון:** צבעים **שטוחים** בלבד, **ניגוד גבוה** (שחור על צהוב, ורוד על שחור, שחור על טורקיז). בלי דרגות ביניים מעודנות.

## טיפוגרפיה (כשיש טקסט בתמונה)

- טקסט פיקסלי בסגנון **`Press Start 2P`** — אותיות בלוקיות, 8-bit, חדות.
- אם נדרש טקסט סריף — בסגנון **`Libre Baskerville`**.
- בלי פונטים סאנס-סריף מודרניים (Inter, Helvetica וכו').
- מומלץ למזער טקסט בתוך התמונה (מודלים מתקשים בכיתוב) — להעדיף אייקונים וצורות.

## מסגרות, קצוות וצללים

- מסגרות **שחורות עבות** סביב אובייקטים (`#1A1A1A` / `#000`), קו עבה אחיד.
- קצוות חדים או מלבנים עם פינות מעוגלות **מתונות** בלבד. בלי עיגול בסגנון iOS.
- **צללים קשיחים בסגנון 8-bit** בלבד (offset שטוח, למשל צל אפור מתחת-מימין). **אין** צללים מטושטשים (blur) ואין gradients רכים.

## קומפוזיציה ואלמנטים דקורטיביים (חלק מהזהות)

- **צורות גיאומטריות מפוזרות** בקצוות הקאדר, חלקן יוצאות חלקית מחוץ למסגרת:
  - חצי-עיגול ורוד (`#E84477`)
  - קשת/ספירלה צהובה (`#F5C800`)
  - קשת טורקיז (`#3BBFB8`)
- מוטיבים חוזרים: מוניטור רטרו עם מסך קרם, מקלדת פיקסלית צהובה, כפתור START אפור, עיגול-מבטא ורוד עם מספר פיקסלי.
- קומפוזיציה נקייה, מרכזית, עם הרבה "אוויר"; הצורות הדקורטיביות ממסגרות את הנושא המרכזי.

## נוסחת prompt מומלצת ליובל

בנה כל prompt משכבות אלו (התאם את "הנושא" לבקשה):

```
<הנושא המבוקש>, retro 1980s-90s pixel-art computer aesthetic, flat colors,
high contrast, thick solid black outlines, hard 8-bit offset shadows (no blur,
no gradients), color palette strictly: black #000000, hot pink #E84477,
teal #3BBFB8, yellow #F5C800, cream #F5F0D0; decorative geometric shapes
(pink half-circle, yellow spiral, teal arc) scattered at the edges, clean
centered composition, vintage desktop-computer vibe.
```

## עשה / אל תעשה

✅ צבעים שטוחים מהפלטה, מסגרות שחורות עבות, פיקסלים חדים, צורות גיאומטריות, ניגוד גבוה.
✅ אסתטיקת מחשב שולחני רטרו (מוניטור, מקלדת, מסך פיקסלי).

❌ אין gradients רכים, אין צללים מטושטשים (blur), אין ריאליזם פוטוגרפי או 3D מודרני.
❌ אין צבעים מחוץ לפלטה, אין פונטים סאנס-סריף מודרניים, אין עיגול קצוות מוגזם בסגנון iOS.
